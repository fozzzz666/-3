1. Механика

1.1 Кинематика

Кинематика изучает движение тел без рассмотрения причин этого движения.

  • Механическое движение — изменение положения тела в пространстве относительно других тел с течением времени.
  • Материальная точка — тело, размерами которого в условиях данной задачи можно пренебречь.
  • Система отсчёта — совокупность тела отсчёта, связанной с ним системы координат и прибора для измерения времени.
  • Траектория — линия, вдоль которой движется тело.
  • Путь (\(s\)) — скалярная величина, длина траектории.
  • Перемещение (\(\vec{s}\)) — вектор, соединяющий начальное и конечное положение тела.

Скорость и ускорение:

  • Средняя путевая скорость: \(v_{ср} = \frac{\Delta s}{\Delta t}\).
  • Мгновенная скорость: \(\vec{v} = \frac{\Delta \vec{r}}{\Delta t}\) при \(\Delta t \to 0\).
  • Ускорение: \(\vec{a} = \frac{\Delta \vec{v}}{\Delta t}\).

Равномерное прямолинейное движение: \(v = const\). Уравнение движения: \(x = x_0 + v_xt\).

Равноускоренное прямолинейное движение: \(a = const\).

  • \(v = v_0 + at\)
  • \(x = x_0 + v_0t + \frac{at^2}{2}\)
  • \(s = \frac{v^2 - v_0^2}{2a}\)

Свободное падение: движение тела под действием силы тяжести. Ускорение свободного падения \(g \approx 9.8 \ \text{м/с}^2\).

Движение по окружности:

  • Центростремительное ускорение: \(a_ц = \frac{v^2}{R} = \omega^2 R\).
  • Угловая скорость: \(\omega = \frac{2\pi}{T} = 2\pi \nu\).
  • Связь линейной и угловой скорости: \(v = \omega R\).

1.2 Динамика

Динамика изучает причины возникновения движения (взаимодействие тел).

Законы Ньютона:

  • I закон: Существуют инерциальные системы отсчёта (ИСО), относительно которых тело движется равномерно и прямолинейно или покоится, если на него не действуют другие тела или их действие скомпенсировано.
  • II закон: В ИСО ускорение тела прямо пропорционально векторной сумме сил, действующих на тело, и обратно пропорционально его массе. \(\vec{a} = \frac{\vec{F}}{m}\) или \(\vec{F} = m\vec{a}\).
  • III закон: Силы, с которыми два тела действуют друг на друга, равны по модулю, противоположны по направлению и имеют одну природу. \(\vec{F}_{12} = -\vec{F}_{21}\).

Силы в механике:

  • Сила тяжести: \(F = mg\).
  • Сила упругости (закон Гука): при малых деформациях растяжения/сжатия сила упругости пропорциональна величине деформации. \(\vec{F} = -k\vec{x}\), где \(k\) — жёсткость тела, \(x\) — деформация.
  • Сила трения скольжения: \(F_{тр} = \mu N\), где \(\mu\) — коэффициент трения, \(N\) — сила нормальной реакции опоры.
  • Сила трения покоя: равна по величине и противоположна по направлению силе, стремящейся сдвинуть тело, и изменяется от 0 до \(F_{тр.покоя, max} = \mu N\).
  • Вес тела (\(P\)) — сила, с которой тело действует на опору или подвес. В ИСО при отсутствии ускорения \(P = mg\).
  • Закон всемирного тяготения: Сила гравитационного притяжения между двумя материальными точками пропорциональна произведению их масс и обратно пропорциональна квадрату расстояния между ними. \(F = G\frac{m_1 m_2}{R^2}\), где \(G = 6.67 \times 10^{-11} \ \text{Н·м}^2/\text{кг}^2\).
→ Задачи по динамике → Тест по механике

1.3 Статика

Статика изучает условия равновесия тел.

  • Момент силы относительно оси вращения: \(M = F \cdot l\), где \(l\) — плечо силы (кратчайшее расстояние от оси до линии действия силы).
  • Условия равновесия твёрдого тела в ИСО:
    1. Векторная сумма всех сил, действующих на тело, равна нулю: \(\sum \vec{F_i} = 0\). (Нет поступательного движения).
    2. Сумма моментов всех сил относительно любой оси равна нулю: \(\sum M_i = 0\). (Нет вращательного движения).
  • Центр масс (центр тяжести) — точка, относительно которой суммарный момент сил тяжести, действующих на все части тела, равен нулю.
  • Гидростатика:
    • Закон Паскаля: Давление, производимое на жидкость или газ, передаётся в любую точку без изменений во всех направлениях.
    • Давление столба жидкости: \(p = \rho g h\).
    • Закон Архимеда: На тело, погружённое в жидкость или газ, действует выталкивающая сила, равная весу вытесненной им жидкости (газа). \(F_A = \rho_{ж} g V_{погр}\).
    • Условие плавания тел: Если \(F_A > mg\) — тело всплывает; \(F_A < mg\) — тонет; \(F_A = mg\) — плавает внутри жидкости.
→ Задачи по статике → Тест по механике

1.4 Законы сохранения

Импульс:

  • Импульс материальной точки: \(\vec{p} = m\vec{v}\).
  • Импульс системы тел: \(\vec{p}_{сист} = \sum m_i \vec{v}_i\).
  • Закон сохранения импульса: В замкнутой системе (сумма внешних сил равна нулю) векторная сумма импульсов тел остаётся постоянной. \(\sum \vec{p}_i = const\).
  • Реактивное движение: Движение, возникающее при отделении от тела с какой-либо скоростью некоторой его части. (Напр., движение ракеты).

Работа и энергия:

  • Работа силы: \(A = F s \cos\alpha\), где \(\alpha\) — угол между вектором силы и вектором перемещения.
  • Мощность: \(P = \frac{A}{t} = Fv \cos\alpha\).
  • Кинетическая энергия: \(E_k = \frac{mv^2}{2}\).
  • Потенциальная энергия:
    • Тела в поле тяжести: \(E_p = mgh\).
    • Упруго деформированного тела: \(E_p = \frac{kx^2}{2}\).
  • Закон сохранения механической энергии: В замкнутой системе, где действуют только консервативные силы (силы тяжести и упругости), полная механическая энергия сохраняется. \(E_{k1} + E_{p1} = E_{k2} + E_{p2}\).
  • Закон изменения механической энергии: Изменение полной механической энергии системы равно работе непотенциальных (диссипативных) сил. \(\Delta E = A_{непот}\).
→ Задачи на законы сохранения → Тест по механике

1.5 Механические колебания и волны

Колебания:

  • Гармонические колебания: Колебания, происходящие по закону синуса или косинуса. \(x = A \sin(\omega t + \varphi_0)\).
  • Характеристики: Амплитуда (\(A\)), период (\(T\)), частота (\(\nu = 1/T\)), циклическая частота (\(\omega = 2\pi / T = 2\pi \nu\)), фаза колебаний.
  • Пружинный маятник: Период колебаний \(T = 2\pi \sqrt{\frac{m}{k}}\).
  • Математический маятник: Период колебаний \(T = 2\pi \sqrt{\frac{l}{g}}\).
  • Вынужденные колебания. Резонанс: Явление резкого возрастания амплитуды вынужденных колебаний при совпадении частоты вынуждающей силы с собственной частотой колебательной системы.

Волны:

  • Поперечные и продольные волны.
  • Длина волны (\(\lambda\)) — расстояние, на которое распространяется волна за один период. \(\lambda = vT = \frac{v}{\nu}\).
  • Звук — продольная волна с частотой от 16 Гц до 20 кГц. Скорость звука зависит от среды.
→ Задачи по колебаниям и волнам → Тест по механике

2. Молекулярная физика и термодинамика

2.1 Основы Молекулярно-кинетической теории (МКТ)

Основные положения МКТ: Вещество состоит из частиц; частицы хаотически движутся; частицы взаимодействуют друг с другом.

Основные величины:

  • Количество вещества: \(\nu = \frac{N}{N_A}\).
  • Постоянная Авогадро: \(N_A = 6.02 \times 10^{23} \ \text{моль}^{-1}\).
  • Молярная масса: \(M = m_0 N_A\).
  • Концентрация: \(n = \frac{N}{V}\).

Модель идеального газа: Собственный объём молекул пренебрежимо мал; между столкновениями молекулы не взаимодействуют; столкновения абсолютно упругие.

Основное уравнение МКТ идеального газа: \(p = \frac{1}{3} n m_0 \overline{v^2} = \frac{2}{3} n \overline{E}_n\), где \(\overline{E}_n\) — средняя кинетическая энергия поступательного движения молекул.

Температура и её связь с энергией:

  • Абсолютная температура \(T\) (в Кельвинах) — мера средней кинетической энергии молекул.
  • \(\overline{E} = \frac{3}{2} k T\), где \(k = 1.38 \times 10^{-23} \ \text{Дж/К}\) — постоянная Больцмана.
  • Уравнение \(p = nkT\).
→ Задачи по МКТ → Тест по молекулярной физике

2.2 Газовые законы. Изопроцессы

Уравнение состояния идеального газа (Менделеева-Клапейрона): \(pV = \nu RT = \frac{m}{M} RT\), где \(R = 8.31 \ \text{Дж/(моль·К)}\) — универсальная газовая постоянная.

Изопроцессы в газах (с постоянным количеством вещества):

  • Изотермический (\(T=const\)): Закон Бойля-Мариотта. \(pV = const\).
  • Изохорный (\(V=const\)): Закон Шарля. \(\frac{p}{T} = const\).
  • Изобарный (\(p=const\)): Закон Гей-Люссака. \(\frac{V}{T} = const\).

Закон Дальтона: Давление смеси химически невзаимодействующих газов равно сумме их парциальных давлений. \(p = p_1 + p_2 + \dots\).

→ Задачи на газовые законы → Тест по молекулярной физике

2.3 Термодинамика

Внутренняя энергия идеального газа: \(U = \frac{i}{2} \nu RT\), где \(i\) — число степеней свободы молекулы (для одноатомного газа \(i=3\), для двухатомного \(i=5\)).

Способы изменения внутренней энергии:

  • Совершение работы (механическое взаимодействие).
  • Теплопередача (тепловое взаимодействие): теплопроводность, конвекция, излучение.

Количество теплоты:

  • При нагревании/охлаждении: \(Q = cm\Delta T\).
  • При сгорании топлива: \(Q = qm\).

Работа в термодинамике: Работа газа \(A = p \Delta V\) (для изобарного процесса). Геометрический смысл работы газа — площадь под графиком процесса в координатах \(p(V)\).

Первое начало термодинамики: Количество теплоты, переданное системе, идёт на изменение её внутренней энергии и на совершение системой работы над внешними телами. \(Q = \Delta U + A\).

Применение 1-го начала к изопроцессам:

  • Изотермический: \(Q = A\).
  • Изохорный: \(Q = \Delta U\).
  • Изобарный: \(Q = \Delta U + A\).
  • Адиабатный (без теплообмена): \(\Delta U = -A\).

Второе начало термодинамики: Невозможен процесс, единственным результатом которого была бы передача теплоты от холодного тела к горячему (формулировка Клаузиуса). Невозможен вечный двигатель второго рода (формулировка Томсона).

Тепловые машины. КПД:

  • КПД тепловой машины: \(\eta = \frac{A}{Q_1} = 1 - \frac{Q_2}{Q_1}\).
  • Цикл Карно — идеальный цикл с максимально возможным КПД. \(\eta_{max} = 1 - \frac{T_2}{T_1}\), где \(T_1\) — температура нагревателя, \(T_2\) — температура холодильника.
→ Задачи по термодинамике → Тест по молекулярной физике

2.4 Фазовые переходы. Влажность воздуха

Фазовые переходы:

  • Плавление / кристаллизация: \(Q = \lambda m\).
  • Парообразование (испарение, кипение) / конденсация: \(Q = L m\).

Насыщенные и ненасыщенные пары. Давление и плотность насыщенного пара зависят только от температуры (и не зависят от объёма).

Влажность воздуха:

  • Абсолютная влажность: \(\rho\) (плотность водяного пара).
  • Относительная влажность: \(\phi = \frac{\rho}{\rho_н} = \frac{p}{p_н} \times 100\%\), где \(\rho_н, p_н\) — плотность и давление насыщенного пара при данной температуре.
  • Точка росы — температура, при которой пар становится насыщенным.
→ Задачи на фазовые переходы → Тест по молекулярной физике

3. Электродинамика

3.1 Электростатика

Электрический заряд. Два вида заряда: положительный и отрицательный. Элементарный заряд \(e = 1.6 \times 10^{-19} \ \text{Кл}\). Закон сохранения заряда.

Закон Кулона: Сила взаимодействия двух точечных неподвижных зарядов в вакууме прямо пропорциональна произведению модулей зарядов и обратно пропорциональна квадрату расстояния между ними. \(F = k\frac{|q_1 q_2|}{r^2}\), где \(k = 9 \times 10^9 \ \text{Н·м}^2/\text{Кл}^2 = \frac{1}{4\pi \varepsilon_0}\).

Электрическое поле. Силовая характеристика — напряжённость (\(\vec{E} = \frac{\vec{F}}{q}\)).

  • Поле точечного заряда: \(E = k\frac{q}{r^2}\).
  • Принцип суперпозиции: \(\vec{E} = \vec{E_1} + \vec{E_2} + \dots\).

Проводники и диэлектрики в электростатическом поле. Диэлектрическая проницаемость \(\varepsilon\).

Потенциальность поля. Потенциал и разность потенциалов.

  • Потенциальная энергия заряда в поле: \(W_p = q \varphi\).
  • Потенциал поля точечного заряда: \(\varphi = k\frac{q}{r}\).
  • Разность потенциалов (напряжение): \(U = \varphi_1 - \varphi_2 = \frac{A}{q}\).
  • Связь для однородного поля: \(U = Ed\).

Конденсаторы. Электроёмкость. Ёмкость плоского конденсатора: \(C = \frac{\varepsilon \varepsilon_0 S}{d}\).

  • Последовательное соединение: \(\frac{1}{C} = \frac{1}{C_1} + \frac{1}{C_2}\).
  • Параллельное соединение: \(C = C_1 + C_2\).
  • Энергия заряженного конденсатора: \(W = \frac{CU^2}{2} = \frac{q^2}{2C} = \frac{qU}{2}\).
→ Задачи по электростатике → Тест по электродинамике

3.2 Законы постоянного тока

Сила тока: \(I = \frac{\Delta q}{\Delta t}\). Условия существования тока.

Закон Ома для участка цепи: \(I = \frac{U}{R}\).

Сопротивление проводника: \(R = \rho \frac{l}{S}\).

Последовательное и параллельное соединение проводников.

Работа и мощность тока:

  • Работа: \(A = UIt\).
  • Мощность: \(P = UI = I^2R = \frac{U^2}{R}\).
  • Закон Джоуля-Ленца: \(Q = I^2Rt\).

ЭДС источника. Закон Ома для полной цепи: \(I = \frac{\mathcal{E}}{R + r}\).

→ Задачи на постоянный ток → Тест по электродинамике

3.3 Магнитное поле

Магнитное поле создаётся движущимися зарядами (токами). Силовая характеристика — вектор магнитной индукции \(\vec{B}\).

Закон Ампера: Сила, действующая на проводник с током в магнитном поле: \(F_A = B I l \sin\alpha\).

Сила Лоренца: Сила, действующая на движущийся заряд в магнитном поле: \(F_Л = |q| v B \sin\alpha\).

Принцип суперпозиции магнитных полей. Правило буравчика (правой руки) для определения направления линий индукции.

→ Задачи на магнитное поле → Тест по электродинамике

3.4 Электромагнитная индукция

Явление электромагнитной индукции — возникновение индукционного тока в замкнутом контуре при изменении магнитного потока через этот контур.

Магнитный поток: \(\Phi = BS \cos\alpha\).

Закон электромагнитной индукции (закон Фарадея): ЭДС индукции в замкнутом контуре равна скорости изменения магнитного потока, взятой с обратным знаком. \(\mathcal{E}_i = -\frac{\Delta \Phi}{\Delta t}\).

Правило Ленца: Индукционный ток всегда имеет такое направление, что его собственное магнитное поле препятствует изменению внешнего магнитного потока, вызвавшего этот ток.

ЭДС индукции в движущихся проводниках: \(\mathcal{E}_i = B l v \sin\alpha\).

Самоиндукция. Индуктивность. ЭДС самоиндукции: \(\mathcal{E}_s = -L\frac{\Delta I}{\Delta t}\).

Энергия магнитного поля катушки: \(W = \frac{LI^2}{2}\).

→ Задачи на индукцию → Тест по электродинамике

3.5 Электромагнитные колебания и волны

Колебательный контур. Формула Томсона для периода свободных колебаний: \(T = 2\pi \sqrt{LC}\).

3.6 Оптика

Законы геометрической оптики:

  • Закон прямолинейного распространения света.
  • Закон отражения света: угол падения равен углу отражения.
  • Закон преломления света: \(\frac{\sin\alpha_1}{\sin\alpha_2} = n_{21} = \frac{n_2}{n_1} = \frac{v_1}{v_2}\). Абсолютный показатель преломления \(n = \frac{c}{v}\).
  • Полное внутреннее отражение.

Линзы. Оптическая сила линзы: \(D = \frac{1}{F}\). Формула тонкой линзы: \(\frac{1}{F} = \frac{1}{d} + \frac{1}{f}\).

4. Квантовая физика

4.1 Фотоэффект

Фотоэффект — явление вырывания электронов из вещества под действием света.

Законы фотоэффекта:

  • Фототок насыщения пропорционален интенсивности падающего света.
  • Максимальная кинетическая энергия фотоэлектронов линейно растёт с частотой света и не зависит от его интенсивности.
  • Для каждого вещества существует красная граница фотоэффекта — минимальная частота (\(\nu_{min}\)), при которой он ещё возможен.

Уравнение Эйнштейна для фотоэффекта: Энергия фотона идёт на совершение работы выхода и сообщение кинетической энергии электрону. \(h\nu = A_{вых} + \frac{mv^2_{max}}{2}\).

Красная граница фотоэффекта: \(\nu_{min} = \frac{A_{вых}}{h}\), \(\lambda_{max} = \frac{hc}{A_{вых}}\).

→ Задачи по фотоэффекту → Тест по квантовой физике

4.2 Атомная физика

Планетарная модель атома. Постулаты Бора:

  • Атом может находиться только в особых стационарных состояниях с определёнными энергиями \(E_n\). В этих состояниях атом не излучает.
  • При переходе атома из одного стационарного состояния в другое излучается или поглощается фотон с энергией \(h\nu = |E_n - E_k|\).

Энергия и импульс фотона: \(E_{\phi} = h\nu = \frac{hc}{\lambda}\), \(p_{\phi} = \frac{h}{\lambda}\).

→ Задачи по атомной физике → Тест по квантовой физике

4.3 Физика атомного ядра

Состав ядра: Протоны и нейтроны (нуклоны). Зарядовое (\(Z\)) и массовое (\(A\)) числа.

Энергия связи ядра: \(\Delta E = \Delta m c^2\), где \(\Delta m\) — дефект масс.

Радиоактивность. Закон радиоактивного распада: \(N = N_0 \cdot 2^{-t/T}\), где \(T\) — период полураспада.

Виды радиоактивных излучений: α-распад, β-распад, γ-излучение.
